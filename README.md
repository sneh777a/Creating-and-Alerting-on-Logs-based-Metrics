# Creating and Alerting on Logs-based Metrics

This repository is created for **practice and learning purposes** while exploring **Google Cloud Monitoring and Logging**.

It contains a Bash automation script that helps set up resources for experimenting with **log-based metrics and alert policies** in Google Cloud.

## Run the Script

Download and run the script using the following commands:

```bash
curl -LO https://raw.githubusercontent.com/sneh777a/Creating-and-Alerting-on-Logs-based-Metrics/refs/heads/main/gcp-gmp-monitoring-setup.sh
sudo chmod +x gcp-gmp-monitoring-setup.sh
./gcp-gmp-monitoring-setup.sh
```

⚠️ This project is intended **only for educational and practice purposes** while learning Google Cloud, Kubernetes, and DevOps automation with Bash.
